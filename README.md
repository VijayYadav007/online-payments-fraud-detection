# online-payments-fraud-detection
A machine learning-based fraud detection system that analyzes transaction data to classify payments as fraudulent or genuine using Random Forest algorithm.

This project detects fraudulent online transactions using a Random Forest Classifier.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NumPy

## Workflow
1. Data Preprocessing
2. Train-Test Split
3. Model Training
4. Model Evaluation
5. Model saved using Pickle

## Result
The model predicts whether a transaction is Fraud or Genuine.
